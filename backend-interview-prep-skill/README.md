# Backend Interview Prep Skill

This Skill keeps Backend Interview Prep sessions consistent across chats.

It defines the teaching methodology, session workflow, and repository usage rules.

## Usage

Start a new chat and say:

```text
Use Backend Interview Prep Skill.
Continue from my repository roadmap.
Start the next day.
```

## Repository Dependency

This Skill expects a backend interview prep repository containing:

```text
AGENTS.md
ROADMAP.md
pattern-library.md
src/main/java/...
```

## What This Skill Does

* Keeps sessions roadmap-driven
* Preserves the guided teaching style
* Ensures pattern recognition before coding
* Ensures state/invariant thinking
* Generates concise notes after each topic
* Avoids random topic drift

## What This Skill Does Not Store

It does not store learning progress.

Progress should stay in:

```text
ROADMAP.md
pattern-library.md
module notes
Java implementations
```
